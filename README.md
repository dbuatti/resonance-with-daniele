# Resonance with Daniele

A joyful, welcoming space for anyone who loves to sing. This application serves as the central hub for the Resonance choir community in Melbourne, providing resources, event management, and member engagement tools.

## Features
- **Member Resources**: Access sheet music, audio tracks, and lyrics.
- **Event Management**: View and RSVP to upcoming pop-up choir sessions.
- **Song Suggestions**: Community-driven repertoire selection.
- **Admin Dashboard**: Comprehensive tools for managing members, marketing, and finances.

Built with React, TypeScript, Tailwind CSS, and Supabase.
